## 📌 Project Overview

The Personal Portfolio Website is designed to represent my professional profile as a Computer Science student and aspiring Web Developer. It provides information about my technical skills, projects, academic achievements, and contact details.

The website features a modern dark-themed user interface with responsive design, hover effects, animations, and smooth navigation for an enhanced user experience.
